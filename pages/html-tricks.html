Hidden Content for Screen Readers

Provide content for screen readers while keeping it visually hidden.

<span class="sr-only">This text is only for screen readers</span>

<style>
  .sr-only {
    position: absolute;
    width: 1px;
    height: 1px;
    padding: 0;
    margin: -1px;
    overflow: hidden;
    clip: rect(0, 0, 0, 0);
    white-space: nowrap;
    border-width: 0;
  }
</style>
Why it works:

This technique visually hides content while keeping it accessible to screen readers, improving accessibility without affecting visual layout.

Browser Compatibility:

Works in all modern browsers.
